# File Count Program in Java

## Description
This Java program counts the number of lines, words, and characters in a file.

## Features
- Uses command-line argument
- Counts words separated by whitespace
- Uses exception handling (try-catch-finally)

## How to Run
javac FileCount.java  
java FileCount input.txt

## Sample Input
Hello world  
This is Java

## Output
Lines: 2  
Words: 5  
Characters: 23
